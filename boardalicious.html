Boardalicious

  • 2023
  • Next.js
  • TypeScript
  • React Query
  • Tailwind CSS
  • Radix UI

Given a Board Game Geek username, the app picks a random game from your collection, for when you can't decide what to play!

Check it out!

For the most part, it's just an app that helps you pick a board game, but there's also a "collection" page so that you had an easy way to see all the games in your collection. In addition, you have the ability to share games with others by opening them up and clicking the share button, which will give you a sharable URL.

Source code in the Github repository.
