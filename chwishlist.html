Chwishlist

  • 2020
  • JavaScript
  • Pipedream

A price drop / wishlist tool for products on Chewy.

Built using Pipedream. A script pulls prices from Chewy's site triggered by a cron, and sends an email containing the 'hot prices' (if the prices are below the thresholds that have been set) as well as the rest of the prices.

More details are in the Pipedream workflow, as well as some instructions on how to set one up for yourself.
